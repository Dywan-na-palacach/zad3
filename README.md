# API Test Script

## Opis

Skrypt `api_test.py` testuje różne endpointy API przy użyciu biblioteki `requests` w Pythonie. Skrypt sprawdza odpowiedzi HTTP oraz kluczowe elementy w odpowiedziach JSON.

## Wymagania

- Python 3.x
- Biblioteka `requests`
